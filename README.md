go-pkg-proxy
============

## Usage
```
Usage of go-pkg-proxy:
  -addr string
    	bind to addr:port (default ":8000")
  -dry-run
    	try loading modules and exit afterwards
  -log-json
    	log JSON objects (instead of plain-text)
  -log-level string
    	set logging level
  -modules string
    	fetch modules from URL (default "file://go.json")
```
